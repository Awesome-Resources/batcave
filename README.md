<h4 align="">Batcave is chat application </h4>
>Batcave client built with [Electron](https://github.com/electron/electron) and Angular2.

## Clone

```
$ git clone https://github.com/talhasch/ling
$ cd ling
```

## Install dependencies

```
$ npm install
```

### Run

```
$ npm start
```

### Package

```
$ npm run release
```
